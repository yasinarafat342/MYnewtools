# 🚀 Mynewtools - Personal Interaction Tracker

**Mynewtools** is a sleek web application designed to help users track and manage their social interactions. Whether it's a phone call, a text message, or a physical meetup, KeenKeeper provides a structured timeline to visualize your social life.

## ✨ Key Features
- **Interaction Logging:** Easily log calls, texts, videos, and meetups.
- **Advanced Timeline Filter:** A professional-grade filtering system to sort interactions by type.
- **Data Visualization:** Built-in analytics using Recharts for visual insights.
- **Modern UI/UX:** Built with a "Utility-First" approach using Tailwind CSS for a clean, responsive look.

## 🛠 Tech Stack
- **Library:** React 19
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Charts:** Recharts
- **Tooling:** Vite
- **Deployment:** Netlify
